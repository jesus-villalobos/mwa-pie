Python Tutor

Python Tutor is a valuable tool for testing your Python Code, and finding what anything that's wrong. You input any code you have and this program will walk you through what Python is doing, step-by-step. It will point out any errors and will help you understand where your code might be going wrong. Use this when Python says your code is not correct.
