Calculator Review

An in-depth analysis of what is really happening in the calculator project.

Acceptable code for this project is: 
def add(x, y):
    """Docstring"""
    added = x + y
    return added

def subtract(x, y):
    """Docstring"""
    subtracted = x - y
    return subtracted

def divide(x, y):
    """Docstring"""
    divided = x / y
    return divided

def divide_whole(x, y):
    """Docstring"""
    return x // y              --> NOTICE: It is acceptable to simply return the desired value

def power(x, y):
    """Docstring"""
    return x ** y              --> NOTICE PT2: Assignment is not necessary

def remainder(x, y):
    """Docstring"""
    return x % y               --> Simple returns are used more often
